v2rayN 是一款面向 Windows 平台的开源网络代理客户端，基于 .NET 开发，支持 V2Ray、Xray 等主流代理核心。凭借简洁的界面、丰富的功能以及持续更新的社区维护，已成为许多 Windows 用户常用的代理管理工具之一。

无论是节点管理、订阅更新，还是路由规则配置，v2rayN 都能够提供较为完善的使用体验，适用于 Windows 10 和 Windows 11 系统。

⸻

软件特色

多核心支持

兼容 V2Ray Core、Xray Core 等主流核心，可根据需求自由切换，满足不同网络配置场景。

一键导入订阅

支持订阅链接导入、二维码导入以及手动添加节点，配置过程更加简单快捷。

智能路由

内置 GeoIP 与 Geosite 路由规则，可根据访问目标自动进行流量分流，提高使用效率。

图形化管理

采用 Windows 原生界面设计，节点管理、连接切换、日志查看等操作更加直观。

开源持续维护

项目长期保持更新，拥有活跃的开发社区，功能不断完善，兼容性持续提升。
