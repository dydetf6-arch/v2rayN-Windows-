v2rayN 是一款专为 Windows 平台打造的开源代理客户端，支持 Xray、V2Ray 等多种代理核心，拥有简洁直观的图形界面和丰富的网络配置功能。凭借稳定的性能、持续更新以及开源社区维护，v2rayN 已成为许多 Windows 用户常用的代理管理工具。(v2rayN)
软件简介
v2rayN 提供图形化操作界面，支持节点管理、订阅更新、代理切换以及路由规则配置，无需复杂命令即可完成日常管理。
软件兼容多种代理协议，可根据不同网络环境灵活配置，适用于 Windows 10、Windows 11 等主流系统，同时也支持部分 Linux 与 macOS 平台版本。(v2rayN中文网)

核心功能
多核心支持
兼容 Xray Core、V2Fly Core 等主流核心，可根据不同需求自由切换，兼顾稳定性与扩展能力。
丰富的协议兼容
支持 VMess、VLESS、Trojan、Shadowsocks、SOCKS、HTTP 等多种协议，满足不同网络配置需求。(v2rayN中文网)
智能路由管理
内置 GeoIP 与 Geosite 路由规则，可根据访问目标自动进行流量分流，并支持自定义规则配置。
简洁易用
支持订阅导入、二维码导入、配置文件导入及手动添加服务器，一键更新节点，降低配置门槛。
开源项目
源码公开，开发者可自由查看、修改及参与项目维护，拥有完善的社区生态和持续更新机制。

使用说明
首次运行软件后，可通过以下方式快速完成配置：
	●	导入订阅链接

完成配置后即可选择节点，并根据需要开启系统代理或 PAC 模式。
